%% Abstract section.
\abstract{Dieses Dokument beschreibt eine Projektarbeit, welche versucht den Prozess der Textverbesserung zu unterstützen in dem Verfahren aus dem Bereich Text Mining und Visualistik kombiniert werden. So wird im ersten Schritt definiert, wie und welche Daten gesammelt werden und und wie diese daraufhin veranschaulicht werden. Dabei wird versucht eine direkte Brücke zwischen Visualisierung und Text zu schaffen,in dem eine visuelle und textuelle Visualisierung kombiniert wird.  } % end of abstract
